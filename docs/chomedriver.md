# chomedriver

Baixar: https://googlechromelabs.github.io/chrome-for-testing/

[LINUX:](https://storage.googleapis.com/chrome-for-testing-public/128.0.6613.119/linux64/chromedriver-linux64.zip)


ls /usr/local/bin
